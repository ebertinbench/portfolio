Active Directory

Cahier des charges

  • Créer plusieurs contrôleurs de domaines Azure Active Directory ainsi qu'un endpoint pour les utilisateurs.
  • Un utilisateur préinscrit doit pouvoir se connecter à sa session, utiliser un navigateur internet
  • Utiliser une base de donnée préinstallée sur le serveur pour développer une application

Livrables livrés

  • Les controleurs de domaines et l'endpoint sont interconnectés. Les rôles FMSO ont étés répartis
  • L'application était fonctionnelle mais n'a pas été déployée. Elle est simplement executée sur la machine utilisateur
  • Plusieurs GPO ont étés modifiées, aucune n'a été rédigée à la main

Compétences développées

  • Bases du fonctionnement de Active Directory
  • modifications des GPO par défaut
  • intéractions avec un endpoint et une bases de données
