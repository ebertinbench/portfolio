Infrastructures Juniper

Contexte

J'ai eu la chance d'obtenir une partie du matériel proposé par Juniper Networks, avec notamment le routeur Juniper SRX 300. Avec lequel j'ai pu expérimenter pour la première fois la configuration manuelle d'appareils réseau.
Cela fut également une de mes premières intéractions avec la documentation technique proposée par le concepteur du matériel

Compétences développées

  • CLI Juniper
  • configurations d'interfaces
  • relai DHCP
  • zones de sécurité
  • Accès distant telnet / SSH
  • fonctionnement chassis cluster (haute disponibilité, redondance)
